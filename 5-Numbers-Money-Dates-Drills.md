# Numbers, Money & Dates — American Drill File

## PART 1: PRICES & FEES

### How Americans Say Money

| Written | Say It | Phonetic | Notes |
|---------|--------|----------|-------|
| $5 | five bucks | [FYVE buks] | Informal |
| $50 | fifty bucks / fifty dollars | [FIF-tee buks] | |
| $500 | five hundred | [FYVE HUN-dred] | Drop "dollars" in context |
| $2,500 | twenty-five hundred | [TWEN-tee FYVE HUN-dred] | Americans say it this way, NOT "two thousand five hundred" |
| $5,000 | five thousand | [FYVE THOU-zund] | |
| $10,000 | ten thousand / ten k | [TEN K] | "k" is very common in business |
| $50,000 | fifty k | [FIF-tee K] | |
| $100,000 | a hundred k / one hundred thousand | [uh HUN-dred K] | |
| $250,000 | two-fifty k / quarter million | [TOO FIF-tee K] | |
| $1,000,000 | a million / a mil | [uh MIL] | Casual business talk |
| $1,200,000 | one point two million | [WUN POYNT TOO MIL-yun] | |
| $10,000,000 | ten million | [TEN MIL-yun] | |

### Work Sentence Drills — Prices

Say each sentence out loud. Stress the CAPITAL syllables.

1. "My starting package is FIVE HUN-dred a month."
2. "For the full setup, we're looking at TWEN-tee FIVE HUN-dred."
3. "Most of my clients pay around TEN K for a three-month engagement."
4. "If we add the API integrations, it goes up to FIF-tee K total."
5. "This system saves you about a HUN-dred K a year in manual work."
